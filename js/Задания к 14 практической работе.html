Реализуйте функцию для решения задачи ЛР3#5789(открыть в новой вкладке)

  • В файле для исходного кода "lab14.js" реализуйте функцию distanceBetweenInts. Она принимает 4 аргумента.
  • В теле функции нужно вернуть результат решения задачи, используя значения аргументов в качестве исходных данных.

Проверьте корректность работы функции с помощью юнит-теста

  • Создайте юнит-тест для проверки задачи и подпишите его словами "вычисление расстояния между точками. Координаты заданы 4 числами"
  • В теле юнит-теста нужно 5 раз вызвать функцию distanceBetweenInts. При каждом вызове в качестве аргументов нужно передавать значения из разных тестов, описанных в задаче.
  • Корректность результатов каждого вызова функции distanceBetweenInts нужно проверить с помощью функции expect в сочетании с функцией toBeCloseTo. Достаточная точность 4 знака после запятой.
  • При открытии файла SpecRunner.html, в нем должна быть зеленая строка "вычисление расстояния между точками. Координаты заданы 4 числами".
